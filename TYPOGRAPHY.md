# Guide Typographique

## Polices utilisées

### Inter
- **Usage** : Police principale pour tout le texte
- **Poids disponibles** : 300, 400, 500, 600, 700, 800
- **Caractéristiques** : 
  - Police moderne et professionnelle
  - Excellente lisibilité sur écran
  - Optimisée pour les interfaces web
  - Letter-spacing négatif pour les titres (-0.02em à -0.03em)

### JetBrains Mono
- **Usage** : Code et éléments monospace
- **Poids disponibles** : 400, 500, 600, 700
- **Caractéristiques** :
  - Conçue spécifiquement pour les développeurs
  - Excellente distinction des caractères

## Hiérarchie typographique

### Titres
- **H1 (Hero)** : 2.8rem, font-weight: 700, letter-spacing: -0.03em
- **H2 (Sections)** : 2.2rem, font-weight: 700, letter-spacing: -0.02em
- **H3 (Cartes)** : 1.8rem, font-weight: 600, letter-spacing: -0.01em

### Corps de texte
- **Body** : 1.5rem, font-weight: 400, line-height: 2.4rem
- **Description** : 1.4-1.6rem, font-weight: 400, line-height: 1.6-1.7
- **Navigation** : 1.4rem, font-weight: 500, letter-spacing: -0.01em

### Éléments spéciaux
- **Logo** : 1.8rem, font-weight: 700, letter-spacing: -0.02em
- **Tagline** : 1.2rem, font-weight: 400
- **CTA Button** : 1.4rem, font-weight: 600, letter-spacing: -0.01em

## Bonnes pratiques

1. **Letter-spacing négatif** pour les titres améliore la lisibilité
2. **Font-smoothing** activé pour un rendu optimal
3. **Line-height** adapté selon la taille du texte (plus petit pour les titres, plus grand pour le corps)
4. **Font-feature-settings** activé pour les ligatures et alternatives contextuelles
